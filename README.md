# ResQMap

ResQMap is an open-source web platform that uses OpenStreetMap data to show emergency services, accessibility information, and disaster-related resources on a single map interface.

The main aim of this project is to help users quickly find important nearby services and make better decisions during everyday navigation as well as emergency situations.

---

## Project Purpose

In many urban and semi-urban areas, information about emergency services, accessible routes, and local resources is scattered across different platforms.
ResQMap brings all this information together into one simple and interactive interface.

---

## Main Features

* Locate nearby hospitals, police stations, and fire stations
* View routes directly on the map
* Filter wheelchair-friendly routes
* Discover local resources such as pharmacies, shelters, and vendors
* Show disaster safe zones and allow incident reporting

---

## Tech Stack

Frontend: React.js with Leaflet
Backend: Node.js with Express
Database: PostgreSQL (PostGIS)
Map Data: OpenStreetMap

---

## Project Structure

ResQMap/

* client → frontend code
* server → backend APIs
* database → schema and configuration

---

## How to Run

1. Clone the repository
2. Install dependencies
3. Configure the `.env` file
4. Run using Docker or npm
